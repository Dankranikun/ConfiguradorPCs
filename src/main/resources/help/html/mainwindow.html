Ventana Principal

Esta ventana muestra una lista de builds de PC almacenadas en la base de datos. Aquí puedes:

  • Ver builds existentes: La tabla muestra los detalles de cada build, incluyendo ID, usuario, RAM, CPU, GPU, Motherboard, PSU y Disco.
  • Seleccionar una build: Haz clic en una fila para ver los detalles en los campos de texto a la derecha.
  • Borrar una build: Usa el botón "Delete build" para eliminar la build seleccionada de la base de datos (se pedirá confirmación).
  • Navegar páginas: Usa los botones "Previous" y "Next" para desplazarte entre las builds paginadas.
  • Crear nueva build: Haz clic en "Create build" para abrir la ventana de creación.

Nota: Asegúrate de seleccionar una build antes de intentar borrarla.
